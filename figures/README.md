# Figures

This directory stores committed PNG diagnostics generated from the lightweight result artifacts.

Primary figure groups:

- `step5_prediction_baselines/`: prediction baseline metrics, confusion matrix, class distribution, log loss, and directional errors.
- `step6_reconstruction_baselines/`: reconstruction scorecard, rate-distortion curve, feature/level/temporal reconstruction error, and derived LOB diagnostics.
- `step7_alignment/`: frozen-latent transfer, rank alignment, compression trade-off, and sample-level diagnostic association.
- `step8_fairness_robustness/`: fair transfer comparison with bootstrap context, raw logistic C-grid behavior, rank sensitivity, and last-snapshot sensitivity.

Figures are visual summaries only. The auditable numeric values live under `results/`.
